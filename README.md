# TZ Currency Tracker 💱

A simple web-based currency exchange tracker and converter focused on Tanzanian Shilling (TZS).

Features
- Live exchange rates fetched from an external API
- Displays multiple currencies against TZS
- Real-time currency converter
- Clean UI with currency flags and formatted values

Technologies Used
- HTML5
- CSS3
- JavaScript (Vanilla JS)
- Exchange Rate API (https://open.er-api.com)

How It Works
- The app fetches live exchange rates using an API with TZS as the base currency
- It dynamically builds:
  - A table showing currency values in TZS
  - A converter for switching between currencies
- Conversion logic:
  - Convert source currency → TZS
  - Then TZS → target currency
